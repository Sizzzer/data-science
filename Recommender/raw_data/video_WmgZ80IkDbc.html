MODULES COVERED IN THIS TRAINING:

Unit 1: Programming in python and r:

Python & R basics.

String and list objects.
Functions & OOPs concepts.

This module will help users to work on messy, incomplete or complex data to make it usable using the below techniques.
Reading CSV, JSON, XML and HTML files using Python.NumPy & pandas for working on large multidimensional arrays and matrices and for Data manipulation and analysis.Scipy libraries to provide mathematical algorithms and convenience functions built on the Numpy extension of Python.Loading, cleaning, transforming, merging, and reshaping data. This project will equip you with all the skills you need to clean your data in Python, from learning how to diagnose your data for problems to dealing with missing values and outliers.

Descriptive statistics Inferential statistics Hypothesis testing Statistical concepts in Python

Introduction to deep learning Understanding neural network through Tensor Flow Convolution & recurrent neural networks

Unit 6: Image classification project

Unit 7: Image classification TV script generation

Unit 8: Big Data
