Neste Tutorial em Ciência de Dados do Python NumPy, discutimos as matrizes de Indexação e Faturamento da Numpy. Nós Aprendemos Indexação Booleana Numpy. O NumPy é o pacote definitivo para computação científica com o Python. Ele contém entre outras coisas: um poderoso objeto de matriz N-dimensional, ferramentas para integrar código C / C ++ e Fortran, funções sofisticadas (broadcasting), álgebra linear útil, recursos de números aleatórios e transformada de Fourier.
O fatiamento básico (0:32) amplia o conceito básico de fatiamento do Python para N dimensões. O fatiamento básico ocorre quando obj é um objeto de fatia (construído por start: stop: notação de passo dentro de colchetes).
Matrizes NumPy Booleanas (8:12) usadas como índices são tratadas de uma maneira diferente inteiramente de matrizes de índice. Os arrays booleanos devem ter a mesma forma que as dimensões iniciais do array que está sendo indexado. No caso mais simples, o array booleano tem a mesma forma.

O Treinamento Essencial em Ciência de Dados da NumPy introduz o início ao intermediário cientista de dados no NumPy, a biblioteca Python que suporta programação numérica, científica e estatística, incluindo aprendizado de máquina. A biblioteca suporta vários aspectos da ciência de dados, fornecendo objetos de matriz multidimensional, objetos derivados (matrizes e matrizes mascaradas) e rotinas para matemática, lógica, classificação, estatísticas e geração de números aleatórios. Jupyter Notebook, uma ferramenta baseada em navegador para criar documentos interativos com código ativo, anotações e até mesmo visualizações como gráficos. Aprenda como criar arrays NumPy, usar instruções NumPy e snippets e indexar, dividir, iterar e manipular matrizes. Além disso, aprenda como plotar dados e combinar arrays NumPy com classes Python e obter exemplos de NumPy em ação: resolvendo equações lineares, encontrando padrões, executando estatísticas, gerando cubos mágicos e muito mais.

Tópicos incluem:

1. Usando o Notebook Jupyter
2. Criando matrizes NumPy de estruturas Python
3. Matrizes de Fatias
4. Usando técnicas de mascaramento e difusão booleanas
5. Plotagem em notebooks Jupyter
6. Juntando e dividindo matrizes
7. Reorganizando os elementos da matriz
8. Criando funções universais
9. Encontrar padrões
10. Construindo quadrados mágicos e cubos mágicos com NumPy e Python
