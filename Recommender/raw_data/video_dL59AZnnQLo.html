Here is how you can start off with Machine Learning using Python and implement it in a project, within a short time span...

First of all, you will need to get hold of the following libraries in Python, which are of utmost importance.
1. Numpy
2. Pandas
3. Matplotlib

For Numpy, I would recommend that you create an account on DataQuest, click on Data Scientist In Python and skip the python basics. This will take u too introduction of Numpy, which u can easily master.

https://app.dataquest.io/…/introducti...

For Pandas, I would recommend that you follow Corey Schafer "Pandas Tutorials".

https://www.youtube.com/watch…

And Lastly for Matplotlib, follow Sentdex, "Matplotlib Tutorial Series".

https://www.youtube.com/watch…

After getting familiarized by the above 3 Python Libraries, you are halfway through...Now its time to start with ML.

Download this particular book..
https://www.lpsm.paris/pageperso/has/...

I think this is the best book on Machine Learning...Now you would be worrying, reading a book can take ages. No need to worry, just read Chapter 1 and 2 carefully along with code provided on github(link given in the book), grasping everything and you will see the magic.

At the end of chapter 2 you can start off with ML projects on Kaggle.

Here is my first Machine Learning Project on kaggle after following the 6 steps mentioned in video:
https://youtu.be/K9Lt1l4pbkw
