Veja como é fácil criar seu primeiro programa de machine learning, do zero, usando Python! Segue abaixo o arquivo (conjunto de dados) utilizado nessa aula: https://www.kaggle.com/dell4010/wine-...

Curso completo de Machine Learning para iniciantes: https://didatica.tech/curso-de-machin...

Resumo e organização dessa aula:
0:06 Objetivo da aula
0:38 Instalação do Python 3
1:00 Quais dados iremos utilizar
3:26 Baixando o dataset dos dados
4:53 Abrindo o arquivo dentro do Python

Esse dataset contém 6.497 linhas (amostras) e 13 colunas (características) de vinhos classificados em 2 categorias: tinto e branco.

Iremos criar um algoritmo de machine learning para aprender a classificar vinhos com base nos dados de treino, e depois iremos verificar a performance com os dados de teste.
