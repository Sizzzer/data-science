Presented in this video is an implementation of Q-learning I did from scratch (including the game world and its visualization).

Here, an agent starts from a random spawn location in the 16x9 grid world where it has to reach the closest blue block (the "destinations"). The life of the agent ends if it reaches a red block (the "danger zones"). The green blocks denote walls through which the agent cannot traverse.

The algorithm learns an efficient policy starting from scratch. Implementation involves a naive Q-table construction approach.

The resulting arrows (and their transparencies) are representative of the aggregated action values learnt at those locations.
